# AskMaira - Asisten Pengetahuan Anda

Aplikasi Chatbot berbasis AI untuk PT. Well Maira Food yang mampu menjawab pertanyaan berdasarkan dokumen SOP (PDF).

## Fitur Utama

- Membaca banyak file PDF sekaligus.
- Mengingat isi dokumen yang diunggah.
- Jawaban akurat disertai sumber referensi dokumen.

## Cara Menjalankan

1. Clone repository ini.
2. Jalankan `npm install`.
3. Buat file `.env` dan masukkan `GEMINI_API_KEY=KUNCI_KAMU`.
4. Jalankan `node server.js`.
